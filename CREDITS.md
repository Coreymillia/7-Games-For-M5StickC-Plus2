# Credits & Attribution

This project builds upon the work of many talented developers and open-source projects. We are grateful for their contributions to the maker and gaming communities.

## üéÆ Original Game Concepts

### Snake
- **Original Creator:** Nokia (1997)
- **Inspiration:** Classic mobile phone game
- **Implementation:** Custom written for M5StickC Plus2
- **License:** Public domain concept

### Tetris
- **Original Creator:** Alexey Pajitnov (1984)
- **Rotation System:** SRS (Super Rotation System)
- **Modern Features:** Hold piece, ghost preview, lock delay
- **Implementation:** Custom written with modern enhancements
- **License:** Game concept in public domain

### Flappy Bird
- **Original Creator:** Dong Nguyen (2013)
- **Inspiration:** Classic mobile game mechanics
- **Implementation:** Custom written for M5StickC Plus2
- **License:** Game mechanic inspiration

### Breakout
- **Original Creator:** Atari/Steve Wozniak (1976)
- **Inspiration:** Classic arcade brick breaker
- **Enhancements:** Endless mode, random patterns, progressive difficulty
- **Implementation:** Custom written
- **License:** Public domain concept

### Space Wars
- **Original Inspiration:** Classic arcade space shooters
- **Based on concepts from:**
  - [M5SpaceWars by volos](https://github.com/volos/M5SpaceWars) - Initial inspiration
  - Galaga, Space Invaders mechanics
- **Implementation:** Heavily modified and enhanced for M5StickC Plus2
- **Features Added:** Progressive difficulty, erratic AI, bullet hell mechanics
- **Original License:** Various open source

### Pong
- **Original Creator:** Atari/Allan Alcorn (1972)
- **Based on:** Classic table tennis game
- **Inspiration:** [m5stickc-plus-2-ping-pong](https://github.com/robouden/m5stickc-plus-2-ping-pong) by robouden
- **Implementation:** Custom AI and physics
- **License:** Public domain concept

### Dodge (Original Game)
- **Creator:** This project (coreymillia + GitHub Copilot CLI)
- **Concept:** Dodge falling Tetris pieces with rotation mechanics
- **Unique Features:** 
  - Block-to-block collision detection
  - Real-time piece rotation
  - Progressive difficulty
- **License:** MIT
